set_automatique(self, automatique)

source code 

Passe le remote en mode automatique ou manuel. En mode automatique, les écrans s'affichent et se remplissent seuls. Permet de tester le programme avec les interfaces graphiques.

set_simulation(self, simulation)

source code 

Passe le remote en mode simulation ou non. En mode simulation les écrans ne s'affichent pas, le remote renvoie directement des valeurs aléatoires. Permet de tester le programme sans les interfaces graphiques.
